# 🛡️ AI Phishing Email Detector

An intelligent web-based application that detects phishing emails using AI/ML techniques.
This project analyzes email content and predicts whether it is **Phishing** or **Safe**, helping users avoid scams and cyber threats.

---

## 🚀 Features

* 🔍 Detects phishing emails in real-time
* 🤖 AI/ML-based prediction system
* 📊 Clean and interactive UI
* ⚡ Fast API response using backend services
* 🔐 Secure handling of user input
* 🌐 Full-stack implementation (Frontend + Backend)

---

## 🧠 How It Works

1. User enters email content
2. Data is sent to backend API
3. AI model analyzes the text
4. Prediction returned:

   * ✅ Safe Email
   * ⚠️ Phishing Email

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/adi-cse/AI-phishing-email-detector.git
cd AI-phishing-email-detector
```

### 2️⃣ Install dependencies

**Frontend**

```
cd Frontend
npm install
npm run dev
```

**Backend**

```
cd Backend
npm install
node server.js
```
